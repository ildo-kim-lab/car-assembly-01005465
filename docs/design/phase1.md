# Phase 1: 자동차 타입 선택

## 목표

사용자가 조립할 차량의 종류(타입)를 선택한다. 현재는 Sedan, SUV, Truck 중에서 선택할 수 있으며, 향후 차량 종류가 추가될 수 있다.

## 입력

메뉴에서 아래 중 하나를 숫자로 입력한다.

| 입력 | 차량 타입 |
|---|---|
| 1 | Sedan |
| 2 | SUV |
| 3 | Truck |

## 처리

- `assembly.py`의 `main()` `step == 0` 분기에서 입력을 받는다.
- `is_valid_range(0, ans)`로 입력값이 1~3 범위인지 검증한다.
- `select_car_type(ans)`를 호출하여 선택 결과를 전역 변수 `q0`에 저장하고, 선택한 차량 타입을 안내 메시지로 출력한다.
- 처리 후 `step`을 1로 진행하여 Phase 2(부품 선택)로 이동한다.

## 출력

- 선택한 차량 타입 안내 메시지 (예: "차량 타입으로 Sedan을 선택하셨습니다.")
- 범위를 벗어난 입력의 경우 에러 메시지 출력 후 동일 단계 재시도

## 관련 코드

- `assembly.py`: `show_menu(0)`, `is_valid_range()`, `select_car_type()`
